# Node postgres

Node + Postgres com Sequelize

## Setup

### Migrate

```bash
./node_modules/.bin/sequelize db:migrate
```

### Dev

```bash
./node_modules/.bin/grunt
```

### Run Watch

```bash
./node_modules/.bin/gulp
```
